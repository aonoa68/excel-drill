# Excel 適応ドリル

北星学園大学「情報入門」の自習用ドリル。
「Excelを訓練したい」学生向けに、**操作の暗記ではなく「いつ・何を・なぜ使うか」の判断力**を鍛える。

公開URL: https://aonoa68.github.io/excel-drill/

## 特徴

- **1ファイル完結**（index.html のみ。HTML+CSS+JS）
- **サーバー不要・ログイン不要**（GitHub Pages で動く静的サイト）
- **適応型出題**：
  - L1 間違えた分野の問題を優先的に出す
  - L2 正解が続けば難易度↑、間違えれば↓
  - L3 間違えた問題はあとで再出題
- 進捗はブラウザの localStorage に保存

## 問題（全20問・5分野）

| 分野 | 問題数 | 内容 |
|---|:---:|---|
| 関数の選択 | 5 | SUM/AVERAGE/COUNT/IF/MIN をいつ使うか |
| 参照 | 3 | 相対参照・絶対参照・複合参照 |
| 数値と文字列 | 3 | 計算できない理由、電話番号の0問題 |
| データ判断 | 5 | 平均と中央値、グラフ選択、集計の目的 |
| 数式の読み書き | 4 | =の意味、範囲指定、SUM vs +、自動化発想 |

## 設計思想

操作手順の暗記問題は意図的に入れていない。
「どの関数を使うか」「なぜ絶対参照が必要か」「このデータはどう見るべきか」など、
**考え方が分かれば、操作はAIや検索で補える**という前提で作っている。

## 更新方法

1. `index.html` の `QUESTIONS` 配列に問題を追加・編集
2. `git add . && git commit -m "..." && git push`
3. GitHub Pages に自動反映
